# encryption-and-the-decryption-tool-
🔐 CipherLab — Encryption & Decryption Toolkit

A medium-difficulty cybersecurity web project built with pure HTML, CSS, and JavaScript. No frameworks, no installation — just open index.html in your browser.

## Features
- 🔒 6 Encryption Algorithms: AES-256, Caesar Cipher, Vigenere, Base64, ROT-13, XOR
- #️⃣ Hash Generator: SHA-256, SHA-1, MD5, SHA-512 (real-time)
- 🛡️ Password Strength Checker with per-criteria feedback
- ⇄ Swap input/output, one-click copy, live status bar
- 📖 Built-in Algorithm Reference Guide

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- CryptoJS 4.1.1 (AES + Hashing)
- Google Fonts (Syne + Share Tech Mono)

## Concepts Covered
- Symmetric encryption (AES, XOR)
- Classical ciphers (Caesar, Vigenere, ROT-13)
- Encoding vs Encryption (Base64)
- Cryptographic hashing (SHA-256, MD5)
- Password security best practices

## Getting Started
```bash
git clone https://github.com/your-username/cipherlab.git
cd cipherlab
open index.html
```
