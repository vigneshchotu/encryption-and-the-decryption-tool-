▶ Algorithm Reference
AES-256
Advanced Encryption Standard. Symmetric block cipher used by governments and militaries. Extremely secure with a strong key.
Caesar
Substitution cipher shifting each letter by a fixed number. Historical cipher, easily broken by frequency analysis.
Vigenere
Polyalphabetic cipher using a keyword. Each letter uses a different Caesar shift. Much stronger than Caesar alone.
Base64
Encoding scheme (not encryption) converting binary data to ASCII. Used widely in data transfer, APIs, and URLs.
ROT-13
Rotates letters by 13. Applying it twice restores original. Symmetric — same operation encrypts and decrypts.
XOR
Bitwise XOR with a repeating key. Foundation of many modern ciphers. Perfect secrecy when key is random and as long as plaintext.
